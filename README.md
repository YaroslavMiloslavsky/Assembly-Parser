# Assembly-Parser
### Installation
- Use Linux (Was tested on Ubuntu 20.04)
- Use git tools to 'git clone [this-repository]
- Run the makefile via 'make' command

### Running
Run ./run [the .asm files] in the terminal.
<br>
The program will generate binary files
